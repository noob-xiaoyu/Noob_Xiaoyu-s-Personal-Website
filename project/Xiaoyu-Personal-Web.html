这是什么？

这是您当前正在浏览的这个个人主页的源代码项目。 它不仅仅是一个静态网站，而是一个从零开始构建的、功能丰富的**现代化单页应用 (SPA)**。

这个项目本身既是展示其他作品的平台，也是一个能够体现作者前端开发能力与设计审美的独立作品。

核心功能与设计亮点

  • 单页应用 (SPA) 架构： 基于原生 JavaScript 实现前端路由，所有页面切换都在主容器内动态加载，无需刷新，体验流畅。
  • 动态视觉效果： 拥有随机视频背景和一键亮/暗色主题切换功能，提供了沉浸式的浏览体验。
  • 玻璃拟态 (Glassmorphism) UI： 广泛应用半透明和背景模糊效果，打造富有层次感和现代感的界面。
  • 高级交互组件： 实现了支持平滑动画和无限嵌套的下拉框 (Accordion)，使内容组织更有条理。
  • 反插件干扰： 采用巧妙的 CSS 技术 (mix-blend-mode) 来反制 Dark Reader 等第三方暗黑模式插件，确保原生设计不被破坏。
  • 完全响应式： 能够完美适配从宽屏桌面到小尺寸手机的各种设备。

技术栈

  • 核心： HTML5, CSS3, 和原生 JavaScript (ES6+)。
  • 布局： 大量使用 Flexbox 和 Grid 进行现代化、响应式的页面布局。
  • 异步处理： 通过 async/await 和 fetch API 实现内容的异步加载。
  • 无框架： 未使用任何第三方前端框架（如 Vue, React），所有功能均为原生实现，以展示扎实的 JavaScript 基础。

总结来说，这个项目是一个集现代前端技术、优秀设计和深度个性化于一体的综合性作品，旨在展示一个美观、实用且技术扎实的个人在线名片。
